# 参考/归档仓库 AGENTS.md 模板

适用于历史资料、参考实现、低频维护项目或没有确认运行时的仓库。

```md
# Repository Guidelines

- Category: reference/archive repository.
- Preserve existing history, notes, and README content.
- Do not add build, test, or deployment automation until an active runtime is confirmed.
- If editing docs, verify headings, paths, links, and examples.
- Do not print or commit secrets.
- Treat missing commands as intentional unless current project files prove otherwise.
```
